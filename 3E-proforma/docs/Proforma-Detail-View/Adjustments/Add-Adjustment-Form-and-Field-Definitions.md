### **Add Adjustment Form and Field Definitions**

The options on this form are used to create an adjustment entry.

![](/img/media/image100.png)
| <strong>Field Name</strong> | <strong>Description</strong> |
|---|---|
| <strong>Type</strong> | Select an Adjustment Type from the drop down list. |
| <strong>Method</strong> | Select an option from this drop-down list to determine how the adjustment is applied. The adjustment method will default to “Proportional by entries on the proforma”. There is a set list of adjustment methods to choose from and the method can be modified from what is defaulted. |
| <strong>Start date</strong> | Select a start date for which time cards will be considered for the adjustment. Only time cards with a date that is on or after the date entered are selected. |
| <strong>End date</strong> | Select an end date for which time cards will be considered for the adjustment. Only time cards with a date that is on or before the date entered are selected. |
| <strong>Include Flat Fee Charges</strong> | Select this check box to include the flat fee charges in the adjustment if the matter has a flat fee agreement.  This field is only available when there is a flat fee charge on the proforma and the <em>Adjust Bill</em> option is set to <strong>Percentage</strong>. |
| <strong>Timekeeper</strong> | Select the timekeeper/fee earner whose fees should be adjusted. |
| <strong>Current Total</strong> | Displays the current proforma total. If you have made other edits to cards that would affect the proforma total but have not clicked on Save &amp; recalc, you may not see the edited value as the current total. |
| <strong>Adjust Bill</strong> | Choose one of the following:<br/><strong>Net bill amount</strong> – Type the total of what the actual bill amount should be. It will automatically calculate the amount of the adjustment in the Amount field.<br/><strong>Amount</strong> – Type the amount of the adjustment. To adjust the bill amount down, enter a minus sign (-) before the amount of the desired adjustment.<br/><strong>Percentage</strong> – Type the percentage of the adjustment. To apply a percentage adjustment that adjusts the bill total down, enter a minus sign (-) before the percent discount desired. For example, -10 in the percentage field will decrease the fees by 10%. |
| <strong>Adjusted Total</strong> | Displays a value based on the amount or percentage entered. It allows you to view how the adjustment impacts the overall Proforma total prior to clicking on <strong>Save adjustment</strong>. |
| <strong>Cancel</strong> | Click to exit this form |
| <strong>Save adjustment</strong> | Click to save your changes and close this form<br/><strong>Note</strong>: This button is inactive until the required fields and an adjustment amount or percentage is entered. |

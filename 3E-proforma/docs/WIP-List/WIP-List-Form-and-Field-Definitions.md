## WIP List Form and Field Definitions

The WIP list displays a list of Matters for which you are the assigned Billing Attorney, or you are working as the assigned Billing Attorney, and where the Matters have work-in-progress (WIP).

![](/img/media/image124.png)
**Note**: See [Navigating the Proforma List](../Getting-Started/Navigating-3E-Proforma---Walkthrough.md#navigating-3e-proforma---walkthrough) for information on using the **Search** and **Sort** fields.

| <strong>Field Name</strong> | <strong>Description</strong> |
|---|---|
| <strong>Start date</strong> | Select a start date to restrict Matter WIP totals for a specific date range. |
| <strong>End date</strong> | Select an end date for the period to which you want to restrict the list of Matter WIP totals. |
| <strong>Apply</strong> | Click to apply date entries to the WIP list. |
| <strong>Include cards from other proformas</strong> | Select this check box to include the total of unbilled cards from other proformas for Matters. |
| <strong>Total WIP</strong> | Displays the hash sum of the WIP in the search results. |
| <strong>Top Matters</strong> | Displays up to 5 top matters by total WIP. The table displays Matter Name, WIP amount, and % of Total WIP. |
| <strong>WIP List</strong> |  |
| <strong>Client</strong> | Displays the name of the matter client. |
| <strong>Matter Number/Matter Description</strong> | Displays the matter number or description. |
| <strong>Bill Group</strong> | Displays the bill group to which the matter belongs. |
| <strong>Currency</strong> | Displays the matter currency. |
| <strong>Total WIP</strong> | Displays the total amount of WIP. |
| <strong>Fees</strong> | Displays the amount of fees. |
| <strong>Costs</strong> | Displays the costs. |
| <strong>Charges</strong> | Displays the charges. |
| <strong>Available Funds</strong> | Displays the amount of available funds. |
| <strong>Generate</strong> | Click to generate a proforma. |

### **System Options**

This tab provides a read-only view of 3E Override / System Option settings that apply to 3E Proforma.

**Note**: The **System Options** tab is only viewable by users that have the **3EProformaAdminRole** assigned to their user in 3E.

![](/img/media/image67.png)
| <strong>Field Name</strong> | <strong>Definition</strong> |
|---|---|
| <strong>3E System Options</strong> | Click to open the <strong>Override/Set System Options</strong> page in 3E. |
| <strong>3E Proforma</strong> | Click to display the system options and overrides available for 3E Proforma. |
| <strong>3E Proforma Security</strong> | Click to display 3E Proforma system options that provide granular control of the application. |
| <strong>Options Grid</strong> |  |
| <strong>Search</strong> | Type search criteria to narrow the list of displayed options. |
| <strong>Option</strong> | Displays the option name. |
| <strong>System Default</strong> | Displays the default system setting. |
| <strong>First Override</strong> | Displays the firm-wide default override setting. |

### **Tracking Proformas by Client, Matter, or Timekeeper**

![](/img/media/image111.png)
| <strong>Field Names</strong> | <strong>Descriptions</strong> |
|---|---|
| <strong>Search Criteria</strong> |  |
| <strong>Track by</strong> | Select an option from this drop-down list to specify how to track proformas, or single proforma (e.g., Client, Matter, Timekeeper, or Proforma).<br/><strong>Note</strong>: Search options and results are different when <strong>Proforma #</strong>is selected, see <a href="#tracking-proforma-by-number"><u>Tracking Proforma by Number</u></a> for further information. |
| <strong>In progress</strong> | Select this check box to track proformas that have been opened and are in the edit, approval, or billing process |
| <strong>Not Started</strong> | Select this check box to track proformas that have been generated but not yet opened. |
| <strong>Completed</strong> | Select this check box to track proformas that have been billed, closed, or deferred. |
| <strong>Start Date</strong> | Select a beginning date for a search period.<br/><strong>Note</strong>: Specifying a date range is recommended if searching includes Completed proformas. |
| <strong>End Date</strong> | Select a end date of a search period. |
| <strong>Apply</strong> | Click to apply search criteria. |
| <strong>Details</strong> |  |
| <strong>Record Details</strong> | Displays details for the record used to track proformas:<br/><strong>Client</strong> – Displays client name an number.<br/><strong>Matter</strong> – Displays matter name and number.<br/><strong>Timekeeper</strong> – Displays timekeeper name, ID, and title. |
| <strong>Proformas Assigned</strong> | Displays the number of proformas in the workflow for the selected client, matter, or timekeeper. |
| <strong>Proforma Progresss</strong> | Displays the current status of the proformas in the workflow for the selected client, matter, or timekeeper (i.e., Completed, In Progress, or Not Started). |
| <strong>Search Results</strong> |  |
| <strong>Proforma #</strong> | The proforma number. Click this link to see the Proforma tracking for this specific proforma. |
| <strong>Owner</strong> | Owner of the proforma. |
| <strong>Status</strong> | Current status of the proforma in the workflow:<br/><strong>Completed</strong> = billed, closed, or deferred<br/><strong>In Progress</strong> = proforma has been opened and is in the edit, approval, or billing process<br/><strong>Not Started</strong> = proforma has been generated but not yet opened |
| <strong>Total</strong> | Displays the proforma total. |
| <strong>Client Name</strong> | Displays the client name. |
| <strong>Client #</strong> | Displays the client number. |
| <strong>Matter Name</strong> | Displays the matter name. |
| <strong>Matter #</strong> | Displays the matter number. |
| <strong>Proforma Date</strong> | Displays the proforma generation date. |
| <strong>Last Viewed</strong> | Date the proforma was last accessed. |
| <strong>Completed Date</strong> | Date the proforma was billed, deferred, or closed. |

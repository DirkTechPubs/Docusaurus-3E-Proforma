### **My proformas**

Selecting the My proformas category displays all proformas for which the user is the owner, and which are not yet submitted.

Proformas with the following statuses are available:

The following actions are available depending on proforma status/sub-status:

| <strong>Action</strong> | <strong>Available with status/sub-status</strong> | <strong>Description</strong> |
|---|---|---|
| <strong>Print Proforma</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent/all sub-statuses<br/>Approval pending | Select this action to view a PDF version of the proforma |
| <strong>Bill Preview</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent/all sub-statuses<br/>Approval pending | Select this action to view a PDF version of the bill as it will appear to the client |
| <strong>Close</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent<br/>Urgent/Returned<br/>Urgent/Rejected | Select this action to close the proforma without saving and remove it from the workflow.<br/><strong>Note</strong>: Close is only available to Proforma Owners. |
| <strong>Submit</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent<br/>Urgent/Returned<br/>Urgent/Rejected<br/>Urgent/Biller returned | Select this action to indicate that all the edits for this proforma have been completed and it should be sent to the biller. |
| <strong>Reclaim</strong> | Approval pending<br/>Urgent/Completed<br/>Completed<br/>Completed/Approved<br/>Completed/Biller review | Select this action to return a proforma back to editing after it was submitted |
| <strong>Forward</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent<br/>Urgent/Returned<br/>Urgent/Rejected<br/>Urgent/Biller returned | Select this action to forward a proforma to another user. |
| <strong>Defer</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent<br/>Urgent/Returned<br/>Urgent/Rejected<br/>Urgent/Biller returned | Select this action to indicate that work on this proforma should be deferred to a later time |
| <strong>Set priority</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent<br/>Urgent/Returned<br/>Urgent/Rejected<br/>Urgent/Biller returned | Select this action to set a proforma as priority. Other users who have the proforma in their list will also see that it is marked as priority. |
| <strong>Comments</strong> | Needs review/all sub-statuses<br/>In review/all sub-statuses<br/>Urgent<br/>Urgent/Returned<br/>Urgent/Rejected<br/>Urgent/Biller returned | Select this action to view/add proforma-level comments. |

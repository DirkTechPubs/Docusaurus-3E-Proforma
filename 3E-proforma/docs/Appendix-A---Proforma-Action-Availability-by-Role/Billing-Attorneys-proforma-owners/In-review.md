### **In review**

Selecting the **In review** category displays all proformas for which the user is the owner and which have been touched/edited already.

Proformas with the following statuses are available:

<table><thead><tr><th><strong>Status</strong></th><th><strong>Sub-status</strong></th><th><strong>Description</strong></th></tr></thead><tbody><tr><td rowspan="4"><strong>In review</strong></td><td>-</td><td>Proformas in this status were opened and/or edited by this billing attorney</td></tr><tr><td>Rejected</td><td>This status is for proformas that were rejected by the approver and returned to the billing attorney and that were opened/edited by this billing attorney</td></tr><tr><td>Returned</td><td>This status is for proformas that were returned to the billing attorney by a biller after they were already submitted and that were opened/edited by the billing attorney.</td></tr><tr><td>Biller returned</td><td>This status is for proformas that were returned to the billing attorney by a biller after they were already submitted, and the biller approves/makes specified edits and that were opened/edited by the billing attorney.</td></tr></tbody></table>

 

The following actions are available depending on proforma status/sub-status:

| <strong>Action</strong> | <strong>Available with status/sub-status</strong> | <strong>Description</strong> |
|---|---|---|
| <strong>Print Proforma</strong> | In review/all sub-statuses | Select this action to view a PDF version of the proforma |
| <strong>Bill Preview</strong> | In review/all sub-statuses | Select this action to view a PDF version of the bill as it will appear to the client |
| <strong>Close</strong> | In review/all sub-statuses | Select this action to close the proforma without saving and remove it from the workflow.<br/><strong>Note</strong>: Close is only available to Proforma Owners. |
| <strong>Submit</strong> | In review/all sub-statuses | Select this action to indicate that all the edits for this proforma have been completed and it should be sent to the biller. |
| <strong>Forward</strong> | In review/all sub-statuses | Select this action to forward a proforma to another user. |
| <strong>Defer</strong> | In review/all sub-statuses | Select this action to indicate that work on this proforma should be deferred to a later time |
| <strong>Set priority</strong> | In review/all sub-statuses | Select this action to set a proforma as priority. Other users who have the proforma in their list will also see that it is marked as priority. |
| <strong>Comments</strong> | In review/all sub-statuses | Select this action to view/add proforma-level comments. |
